# Nexus

A modern real-time communication platform built with Spring Boot and Python AI.

## 📋 Prerequisites

This project builds upon the following repositories:
- **Docker Containers** : Install the redis,postgres,kafka,mongo,minio before running the social-core
    - https://github.com/vivekPandeyDev/docker-container

## 🚀 Features

### 💬 Messaging
- One-to-one chat
- Group chat
- Temporary (ephemeral) chat
- Message reactions
- Read receipts
- Typing indicators
- Media sharing

### 🎥 Reel Streaming
- Upload reels
- Infinite scrolling feed
- Like and comment
- Share reels
- Recommendation engine (planned)

### 🤖 AI Assistant
- AI conversations
- Chat summarization
- Smart replies
- Translation
- Content moderation
- Retrieval-Augmented Generation (RAG) (planned)

### 🔒 Security
- JWT Authentication
- OAuth2 (planned)
- Role-based access control
- End-to-end encryption (planned)
